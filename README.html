tiff_stack2avi.exe

tiff_stack2avi.exe converts multiple tif sequences to avi files using ffmpeg. Intended to work on tif sequences generated by fastCamRecord.

How to use

  1. You first need to install ffmpeg on your computer.
  2. Run the program.
  3. Choose the folders that contain tif sequences. The tiff file should be named frame%d.tif.
